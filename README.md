## 🚀 Project Overview

ResidentOne is a **Unified Hostel Management Super App** built to eliminate fragmented hostel systems and provide a seamless, centralized experience for residents and wardens.

This project was developed as part of the **App Development Hackathon by IADC IIT (ISM) Dhanbad in association with Amber Hostel**.

As highlighted in the problem statement, hostel operations today are scattered across multiple platforms — complaints, mess updates, notices, and bookings — making it inefficient for both residents and wardens fileciteturn0file0.

ResidentOne solves this by bringing everything into a **single, intuitive mobile application**.

---

## 📱 Features Implemented

### 👤 Authentication & Profile
- Secure login system
- Role-based access (Admin / Resident)

### 🏠 Dashboard
- Personalized overview
- Quick access to important modules

### 🛠 Complaint Management
- Raise complaints
- Track status in real-time
- Admin resolution panel

### 🍽 Mess Feedback System
- Submit meal feedback
- View ratings

### 📢 Announcements
- Admin can broadcast notices
- Residents receive updates instantly

### 📞 Call Directory
- Easy-to-use contact list
- One-tap calling (elderly-friendly UI)

### 🏋️ Amenities Management
- Add amenities (Admin)
- Booking requests management
- Time slots & scheduling

### 🧑‍💼 Admin Dashboard
- Central control panel for:
  - Complaints
  - Amenities
  - Announcements
  - Feedback

## 🛠 Setup Instructions

1. Clone the repository:
```bash
git clone https://github.com/Aryan181206/HostelX
```

2. Navigate to project:
```bash
cd HostelX
```

3. Install dependencies:
```bash
flutter pub get
```

4. Run the app:
```bash
flutter run
```
